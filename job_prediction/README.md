
# Fake Job Post Detection using Machine Learning

This project detects **fake vs real job postings** using NLP and Machine Learning.

## Technologies
- Python
- Flask
- Scikit-learn
- TF-IDF
- Logistic Regression

## Dataset
Sample dataset located in:
data/jobs_sample.csv

## Train Model
python train_model.py

## Run Web App
python app.py
